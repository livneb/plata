Historian — seed the graph

Asks the LLM to enumerate dramatic market-moving events from 2005-2025 (wars, crises, central-bank surprises, hacks, regulation). For each event it pulls real OHLCV bars from Bybit for the affected symbols and computes price-impact metrics (max move, time-to-max, drawdown). Stored as event nodes in the knowledge graph so the Strategist can compare live events to them.
