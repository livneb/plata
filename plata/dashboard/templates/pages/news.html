Sources

GDELT query

Boolean expression sent to the GDELT Doc API. Use OR/AND, quote phrases. Edits apply on the next poll (~15 min).

Reddit subreddits

One per line, no r/ prefix.

RSS feeds

One per line. Format: Friendly name | https://feed/url. URL alone is fine too.

Telegram channel IDs

One numeric chat ID per line (channels/groups are negative, e.g. -1001234567890). Leave empty to accept all chats the bot is in.

📈 Market ticker

Tracks live prices for the symbols below and emits a PRICE_ACTION signal when |% change| over the window crosses the threshold. Cryptocurrencies use CoinGecko (free, no auth). Stocks use Alpaca data API (requires Alpaca creds in /settings/?tab=api). Signals bypass the news content filter since they're deterministic price events, not news.

Content filter

Applied to every story before it reaches the strategist. Case-insensitive substring match against the title+body.
